# SystemLibrary Common Framework (Source)

Source code for SystemLibrary Common Framework.

## Requirements
- .NET 10 or higher

## Build
Open any solution, e.g. `SystemLibrary.Common.Framework.sln` in Visual Studio and build.

## Contributing
- Create branches using prefix `/task/<desc>`
    - Example: `/task/cache-fallback-does-not-throw-on-null`
- Do not modify version or release notes manually
- Run all tests within the `module` you've changed before creating a PR against `main`
